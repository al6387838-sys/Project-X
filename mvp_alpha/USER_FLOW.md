# User Flow — Primeira Experiência

1. **Aterrissagem**: O usuário acessa a plataforma e é recebido com a proposta de valor clara ("Seu copiloto de vida pessoal").
2. **Educação Rápida**: 3 telas curtas explicando os pilares (Aprende com você, Antecipa o futuro, Age por você, 100% privado).
3. **Setup do Life Profile**: Preenchimento rápido de dados essenciais, valores (até 5) e autoavaliação das áreas da vida.
4. **Conexão de Fontes**: Tela focada em integrar pelo menos 2 fontes de dados (ex: Calendar e Banco) para alimentar o motor.
5. **Geração do Grafo**: O sistema processa os dados e exibe visualmente a criação do Life Graph.
6. **Primeiro Briefing**: O usuário recebe seu primeiro Morning Briefing personalizado, mostrando o valor imediato da plataforma.
7. **Interação com Companion**: Primeira conversa com a IA, que já utiliza o contexto recém-criado.
8. **Primeira Missão**: O usuário é guiado a criar seu primeiro objetivo prático.
9. **Dashboard**: O usuário chega à central de comando, com todas as métricas e módulos ativados.
