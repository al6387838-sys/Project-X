FAQ

Perguntas frequentes

O LifeOS é seguro? Como meus dados são protegidos?
Sim. Utilizamos criptografia AES-256 para dados em repouso e TLS 1.3 para dados em trânsito. Somos certificados SOC 2 Type II e seguimos as diretrizes da LGPD. Seus dados nunca são vendidos ou compartilhados com terceiros.
Posso cancelar a qualquer momento?
Sim, sem fidelidade. Você pode cancelar sua assinatura a qualquer momento diretamente no painel de configurações. Seus dados ficam disponíveis por 30 dias após o cancelamento para exportação.
A IA do LifeOS usa meus dados para treinar modelos?
Não. Seus dados pessoais nunca são usados para treinar modelos de IA. Os insights são gerados de forma isolada para cada usuário, sem compartilhamento de informações entre contas.
Funciona offline?
Sim. O LifeOS é construído com arquitetura offline-first. Você pode registrar hábitos, tarefas e notas sem conexão. Quando a internet voltar, tudo sincroniza automaticamente.
Existe versão mobile?
Sim. O LifeOS está disponível para iOS e Android, com experiência nativa completa. Também funciona como PWA (Progressive Web App) em qualquer navegador mobile.
Como funciona o plano Enterprise para equipes?
O plano Enterprise permite criar organizações com múltiplos workspaces, RBAC granular, SSO corporativo e auditoria completa. Ideal para empresas que querem implementar uma cultura de alta performance.
